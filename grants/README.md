# Aegis Grant Packet

This folder contains grant-readiness material for Aegis-GPU.

Current packet:
- `aegis-project-summary.md`: one-page project summary
- `aegis-outcomes-and-metrics.md`: measurable outcomes and evaluation plan
- `aegis-pilot-plan.md`: pilot structure, timeline, and evidence plan
- `aegis-budget-framework.md`: draft budget narrative and cost buckets
- `aegis-loi-workforce-development.md`: founder-usable grant LOI draft
- `aegis-executive-abstract.md`: short portal and outreach summary text
- `aegis-target-opportunity-scc6.md`: current live target analysis for DOL SCC6
- `aegis-loi-scc6-partner-version.md`: partner outreach draft for a community-college-led SCC6 application
- `aegis-scc6-outreach-email.md`: cold outreach email drafts for college leads
- `aegis-scc6-outreach-short.md`: LinkedIn, warm intro, and follow-up outreach text

Positioning note:
- These materials are written to support founder-led grant applications now.
- They are strong enough for early outreach and small-to-mid-size grant drafting.
- They are not a substitute for a compliance review on a specific funding program.

Current stance:
- Aegis should be presented as a credible AI infrastructure learning and troubleshooting training platform with verified browser-proof evidence and a defined pilot path.
- Aegis should not yet be presented as a market-ranked or independently validated top-5% product.
