- Extended `frontend/js/cluster-dashboard.js` to render:

- Extended `frontend/js/runtime.js` to:

  - log those fault actions into the existing terminal
- Extended `frontend/js/cluster-terminal.js` so terminal output now exposes degraded evidence:

  - `ibstat` reflects degraded fabric state
  - `nvidia-smi topo -m` warns when NVLink health is no longer clean
- Added styling in `frontend/css/styles.css` for:
  - node note chips

  - clear-all action button
- Expanded test coverage:
